## Краткое описание работы
В ходе данной лабораторной работы были изучены основы взаимодействия JavaScript с DOM-деревом. Мы научились находить элементы на странице, изменять их содержимое и стили, обрабатывать действия пользователя (клики, ввод текста, отправку форм)

## Что изучили
*   Что такое DOM (Document Object Model) и как с ним работать через объект `document`.
*   Методы поиска элементов: `getElementById()`, `querySelector()`.
*   Изменение контента (`textContent`) и CSS-стилей (`style`).
*   Обработка событий с помощью `addEventListener()`.
*   Взаимодействие с формами: получение данных из `input`, отмена стандартной отправки (`preventDefault`).
*   Валидация пользовательского ввода и вывод сообщений об ошибках.
*   Динамическое создание HTML-элементов через `createElement()` и `appendChild()`.
*   Делегирование событий для работы с динамически добавленными элементами.



## Сравнение JavaScript (DOM) с C# (WinForms/WPF)

В ходе работы была проведена параллель между веб-технологиями и десктопной разработкой на C#. Концепции оказались очень похожими, отличаясь лишь синтаксисом и средой выполнения.

### Сравнение основных операций

| Концепция | C# (WinForms) | JavaScript (DOM) |
| :--- | :--- | :--- |
| **Найти элемент** | `Button myButton = this.Controls["myButton"];` | `const myButton = document.getElementById("myButton");` |
| **Изменить текст** | `label1.Text = "Новый текст";` | `label1.textContent = "Новый текст";` |
| **Добавить обработчик** | `button1.Click += HandleClick;` | `button.addEventListener("click", handleClick);` |
| **Создать элемент** | `Button btn = new Button();` | `const btn = document.createElement("button");` |
| **Добавить в контейнер** | `panel1.Controls.Add(btn);` | `panel.appendChild(btn);` |
| **Скрыть элемент** | `button1.Visible = false;` | `button.style.display = "none";` |
